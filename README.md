# thx.promise

A typed and simple to use Promise library for Haxe. You can also use it to make cross-target requests (`GET`, `POST`, `PUT`, etc.) across all targets.

# Installation

`thx.promise` isn't available as a haxelib yet. To install it, clone the repository, and run `haxelib dev thx.http .` from the cloned directory.

# Usage

Refer to the [Request class tests](https://github.com/fponticelli/thx.http/blob/master/test/thx/http/TestRequest.hx) source code to see how you can make web requests.
